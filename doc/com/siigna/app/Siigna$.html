  27. def paint (graphics: Graphics, transformation: TransformationMatrix): Unit

    The entrance to the paint-functions of the interfaces, i.

    The entrance to the paint-functions of the interfaces, i. e. the modules, and the Display. For the modules the matrix is forwarded in case the module needs to use/reverse some of the transformations that already have been applied to the view.
    The painting eludes the normal event-based thread communication, since the Control mechanism runs on it's own thread and we'd like to make sure that the painting happens instantly.
