ModuleInterface

class ModuleInterface extends Interface

Every Module is given an unique instance of ModuleInterface to access graphical stuff. The graphical settings are inherited in a direct line from the Siigna object.
A ModuleInterface can be chained to another interface. If the interface is chained every paint-call is forwarded to the chained instance as well. In this way every active module gets painted, starting from the first one initialized to the latest.
