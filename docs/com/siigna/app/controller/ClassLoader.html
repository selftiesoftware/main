  41. def loadModule [T] (classPath: String, filePath: String)(implicit arg0: Manifest[T]): Option[T]

    Load a module class with the given name from a given location.

    Load a module class with the given name from a given location. Note: Siigna tries to locate the class from specific paths in the file. Ordered, they are:

    • com.siigna.module ("com.siigna.module.<moduleName>." exact)
    • module ("module.<moduleName>." exact)
    • name of the module ("<moduleName>." exact) and
    • root ("" exact)

    If you wish to add your own path.

    classPath

    the path to the module to load.

    filePath

    the path to the module as file or URL.

    returns

    Option[T] Some[T] if the module loads successfully and the resulting class is an instance of T, else None.
