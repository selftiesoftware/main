Interface

trait Interface extends AnyRef

An Interface provides functionality to interact with the graphical part of Siigna. There are two types interfaces in the Siigna core: ModuleInterfaces and the Siigna object. The hierarchy Each module is thus as follows: 

      Interface
    ____/   \____
   /             \
Siigna      ModuleInterface
Each module is connected to their unique instance of a ModuleInterface where they can set stuff like cursors, displays, paint-functions etc.
The Siigna object is the main entry-point to the graphical parts of Siigna. It communicates directly to the view. See the Siigna object for more info.
